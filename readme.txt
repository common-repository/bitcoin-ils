Enables site owners to implement the Bitcoin ILS exchange rate on their WP sites.

== Description ==

Bitcoin ILS is a light-weight plugin for WordPress users that want to show the Bitcoin conversion rate to the Israeli Shekel (ILS).
Implemented easily by using the shortcode [BILS] wherever you want the table to appear.

This plugin uses the open API by https://www.coindesk.com
You can read the terms of use here: https://www.coindesk.com/terms-conditions
